## 第二十八章·唤醒的代价

我从楼梯口走出来，脚步在金属台阶上敲出沉闷的回响。

但我没有立刻往B2层走。

在最后一阶台阶上，我停住了。

不是犹豫。

是感知。

我的胸口——「???」安静地蛰伏在那里，像一颗沉入深海的石子。但如果仔细去听，能感觉到它微弱的脉动，像是某种古老生物的心跳。

它还在。

只是很累。

"你还好吗？"我问。

沉默持续了三秒。然后是一阵微弱的涟漪，从胸口蔓延到四肢，像是某种回应。

"……在同步。"它说，声音沙哑得几乎听不清，"和那个机器。"

"什么？"

"E-032。"「???」停顿了一下，像是在组织语言，"唤醒协议激活的时候，它和整个阵列建立了连接。我被卷进去了。"

我皱起眉。

"有影响吗？"

"不知道。"它说，"但现在……感觉好多了。应该是同步完成了。"

"同步完成意味着什么？"

"意味着我能感知到阵列的状态了。"「???」的声音渐渐清晰起来，"几百个休眠舱，都在等待。但有一个……"

"E-032。"

"对。"它说，"它已经在启动了。光柱在消散，但唤醒进程还在继续。"

我的心跳漏了一拍。

"它醒了？"

"还没有完全醒。"「???」说，"光柱是封印，解封之后需要一个过程。但它的意识已经在运转了，像一台刚按下启动键的电脑，还在加载系统。"

我深吸一口气，靠在墙上。

那就还有时间。

至少不是从头开始。

"还有别的吗？"

"有。"「???」说，"阵列深处有个光点，在闪。比之前更亮了。还有……四个休眠舱在嗡鸣。"

四个。

污染先驱者。

"他们的状态？"

"很稳定。"「???」说，"污染指数几乎归零了。应该是在休眠中自我修复。"

我点了点头，转身继续往B3层走去。

这一次，我的目标很明确。

不是探索。

是确认。

---

中央机器还在运转，但界面变了。

```
先驱者协议系统 v3.7
继承者确认中
唤醒协议状态：激活中
E-032唤醒进度：3%
```

3%。

光柱确实在消散了。

我的目光落在屏幕底部，那里有一行小字，是之前没有的：

```
中央机器同步率：34%（待激活）
备注：休眠舱阵列同步率与中央机器同步率为独立系统
```

我愣了一下。

两套系统。

两套同步率。

"这个'独立系统'是什么意思？"我问。

「???」沉默了一瞬。

"意思是……你之前用钥匙激活的是休眠舱阵列系统，那个同步率是给E-032用的。"它说，"现在是中央机器系统，要执行唤醒协议，需要单独的同步率。"

"96%呢？"

"那是阵列同步率，用来唤醒E-032的。"「???」的声音有些疲惫，"两个不同的协议，不同的阈值。96%已经足够让E-032苏醒了，但唤醒协议需要40%。"

我盯着那行字看了很久。

"所以我需要同时满足两套系统的要求？"

"对。"「???」说，"阵列同步率96%，中央机器同步率34%。前者用来唤醒创造者，后者用来唤醒所有人。"

我揉了揉太阳穴。

又是选择。

又是代价。

"差的那6%呢？"我问，"从34%到40%。"

"收集更多记忆碎片。"「???」说，"你回收的每一段记忆，都会同时提升两套系统的同步率。只是阈值不同，感受不到那么明显的波动。"

"那我现在的96%呢？"

"也会继续提升。"「???」说，"只是已经没有阈值限制了。E-032已经醒了。"

我沉默了。

这就说得通了。

第27章的96%是阵列同步率，用来唤醒E-032。第28章的34%是中央机器同步率，用来执行唤醒协议。两条线，两个目标，两个阈值。

数值不再矛盾。

"还有一件事。"我说，"第27章的时候，黑盒子在触碰E-032的时候剧烈震动。现在呢？"

「???」沉默了两秒。

"黑盒子……在同步休眠。"

"什么意思？"

"意思是它完成了自己的任务。"「???」说，"它触发了阵列共振，然后又和阵列建立了连接。现在它和那些休眠舱一样，也在等待。只是它的'休眠'不是睡着，是……保持连接。"

"它还在发信号吗？"

"微弱了。"「???」说，"和阵列同步之后，它的能量被分散了。单点信号几乎探测不到，但如果阵列被激活……它会被重新唤醒。"

我点了点头。

黑盒子没有消失，只是进入了低功耗模式。

这也是一种等待。

---

我转身离开中央机器，沿着休眠舱阵列的边缘走。

果然。

在阵列的深处，我看到了那四个休眠舱。

它们排成一排，外壳上的纹路暗淡了不少。如果仔细听，能听到舱体内部传来的微弱嗡鸣声，像是某种机器待机的频率。

污染指数几乎归零。

他们在休眠中完成了自我修复。

"别唤醒他们。"

我想起了那个刻在休眠舱上的警告。

"你觉得那个警告还有效吗？"我问。

「???」没有立刻回答。

过了几秒，它才开口：

"那个警告是针对三十年前的唤醒协议的。"它说，"当时没有'污染指数'这种东西，也没有'净化协议'。唤醒就是唤醒，成功或失败，没有中间地带。"

"现在呢？"

"现在有缓冲了。"「???」说，"唤醒失败的人会被污染指数标记，然后被强制休眠，等待净化或……转化。"

"转化是什么？"

"变成残骸守护者那种东西。"「???」的声音低下去，"像刘建国那样。"

我停下脚步。

刘建国。

三十年前第一批觉醒者。

被污染后变成残骸守护者，守护这片残骸三十年。

"所以那个警告的意思是……"

"三十年前的唤醒协议不完善。"「???」说，"现在应该好一些了，但代价还是会有的。只是没那么极端。"

"比如？"

"比如失去部分记忆。"「???」说，"比如醒来之后不认识自己。比如……"

它停顿了一下。

"比如什么？"

"比如醒来之后，发现等了自己三十年的人已经不在了。"

我的胸口突然很闷。

那个女先驱者。

她刻下警告的时候，应该是知道会有代价的。但她还是选择了等待，选择了沉睡三十年，只是为了等我回来。

"她还在吗？"我问。

"那个光点？"「???」沉默了一瞬，"还在。只是……形态变了。"

"什么形态？"

"她把自己融进了阵列。"「???」说，"作为唤醒协议的……担保。没有她，这个协议可能早就失效了。"

"所以她一直在运行。"

"对。"「???」说，"她没有睡。她一直在看着。"

我站在休眠舱阵列前，看着远处那个微弱的光点。

它在闪烁，像是一只疲惫但不肯闭合的眼睛。

"我会回来的。"我说。

光点微微颤动了一下，像是在回应。

然后我转身离开。

---

下午两点四十分，我回到了B2层。

李思雨站在楼梯口，看到我的时候明显松了口气。

"怎么样？"

"活着。"我说，"而且有进展。"

我从口袋里摸出手机——这玩意儿居然还有电——给她看了我拍下的照片：中央机器的屏幕、屏幕上显示的同步率、以及那个"唤醒协议状态：激活中"。

李思雨看完之后沉默了很久。

"34%。"她说，"差6%就能完全解锁唤醒协议。"

"但这6%不是靠时间能刷出来的。"我说，"需要找到更多记忆碎片，或者找到别的方法。"

"同步率每提升一点，你回收的记忆就会多一点。"李思雨皱着眉分析，"那个光点女人给你的信息应该也算在里面。还有那个'别唤醒他们'的警告——那也是三十年前留下的记忆痕迹。"

"你觉得那个警告是什么意思？"

"不好说。"李思雨摇头，"但能在休眠舱上刻字，说明她当时还有行动能力。也许她在最后时刻想警告后来的人，也许……她是在警告你。"

"警告我什么？"

"唤醒是有代价的。"李思雨看着我，"如果代价是可承受的，她不会特意留下这个警告。所以——"

"所以她在说，唤醒可能会让一些人出问题。"我接上她的话。

"对。"李思雨点头，"但她同时又在等你回来。这两件事不矛盾。"

"矛盾吗？"

"她可能知道唤醒的代价，但她也知道，有些人不愿永远沉睡。"李思雨说，"就像王德发的妻子。她选择了休眠，但她也留下了那句话——'不是遗忘，是拒绝'。她在拒绝什么？也许是拒绝系统给她的版本，也许是拒绝在不知情的情况下醒来。"

"所以她宁愿沉睡，也要在醒来之前知道真相。"

"但她没办法自己决定。"李思雨的声音低下去，"因为只有你能激活唤醒协议。"

我靠在墙上，感觉疲惫从骨头缝里往外渗。

"那个光点女人，"李思雨突然问，"她是什么身份？"

"三十年前的先驱者，和我一起的。"我说，"她选择了休眠，等待我回来。"

"和你一起的。"李思雨重复了一遍，"那你们以前……"

"不知道。"我打断了她，"想不起来。同步率不够。"

李思雨没有追问。

但她看我的眼神有些奇怪。

"怎么了？"我问。

"96%。"她突然说。

"什么？"

"第27章的时候，"李思雨看着我，"你的同步率飙到了96%。那是什么感觉？"

我愣了一下。

96%。

那是一种……被淹没的感觉。

记忆像潮水一样涌进来，三十年前的画面在脑海里炸开。我看到了自己站在石台前，看到了那个男人转身离开，看到了蓝光从石台里涌出——

"很疼。"我说，"像是有人把另一个人的人生硬塞进你脑子里。"

"那你还好吗？"

我看着李思雨，她的声音有些紧。

"你担心我？"

"我见证了那个过程。"李思雨说，"41%到96%，只用了几分钟。中间还夹杂着记忆洪流、身份冲击……正常人早就疯了。"

我笑了笑。

"谁说我正常了？"

李思雨愣了一下，然后也笑了。

"确实不像。"她说，"正常人不会在听到'我就是那个人'之后，还能站在这儿跟我开玩笑。"

"那是因为没时间崩溃。"我收起笑容，"还有一堆事要处理。"

"王德发呢？"

我沉默了一下。

王德发。

他还在等他的妻子。

"先别告诉他。"我说，"等我把情况理清楚再说。"

李思雨看了我一眼，没有反对。

"三点钟了。"她说，"其他人应该快醒了。"

我点了点头，从墙上撑起身来。

"走吧。"我说，"还有一堆事要处理。"

我走出B2层的设备间，外面是灰蒙蒙的天空。太阳还藏在云层后面，洒下来的光线昏黄暗淡。

末日第二天的下午，刚刚开始。

但我没有立刻往营地走。

在门口站了一会儿，我掏出手机，又看了一眼那张照片——E-032的唤醒进度：3%。

"继承者。"我低声说，"你到底在等什么？"

手机屏幕闪了一下。

然后，界面变了。

```
先驱者协议系统 v3.7
继承者确认中
唤醒协议状态：激活中
E-032唤醒进度：3%
警告：检测到外部信号干扰
```

外部信号干扰。

观测者又来了。

我收起手机，看向远处的天空。灰蒙蒙的云层后面，似乎有什么东西在闪烁。

"别担心。"「???」的声音在脑海里响起，"信号很微弱，还在安全范围内。但……"

"但什么？"

"唤醒完成之后，就不一定了。"

我深吸一口气，转身往营地走去。

同步率34%，唤醒进度3%。

还有一堆事要处理。

但至少，不是从头开始。
