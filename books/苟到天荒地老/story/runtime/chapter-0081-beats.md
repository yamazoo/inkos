## 节拍说明

### 节拍1（悬念引入）
**承接上章结尾**：月光照在脸上/泪痕干涸/暗道尽头
- **场景**：陈守一在昏迷边缘苏醒，模糊视线中看到赵猛守在洞口的身影，耳边传来幽冥兽的嘶吼从远处变得清晰
- **情绪**：从虚无中挣扎回来→感知到危险→试图开口却发不出声
- **关键细节**：手中紧握周沉的玉牌碎块

### 节拍2（局势升级）
**因果反噬机制浮现**
- **场景**：陈守一在赵猛呼唤中彻底清醒，赵猛将水囊递来时，他开始回忆彼岸世界的真相
- **代价揭露**：三源合一后，他明白了——
  - 每平衡一年 = 消耗一年寿元
  - 若失衡 = 双倍因果反噬
  - 母亲承受了四十三年的因果反噬
- **情绪**：从震惊到沉重→计算自己的四年还能做什么
- **势力影响**：这个机制暗示整个大陆的命运

### 节拍3（冲突爆发）
**末法真相完整揭露 / 母子因果真相大白**
- **核心内容**：
  1. **末法时代真相**：三百年前人类的抉择——用灵气彻底稀薄化换取幽冥界永固封印
  2. **因果反噬完整机制**：封印平衡需要代价，母亲是第一个承受者，四十三年独自扛起
  3. **母亲英年早逝真相**：不是病逝，是被因果反噬侵蚀生机
- **情感冲击**：陈守一在赵猛面前无声流泪——母亲从未向他诉苦，独自承受四十年
- **关键对话**：赵猛察觉异样但没有追问，只是默默递上手帕

### 节拍4（小高潮）
**情感冲击：母亲的完整故事**
- **代价**：理解母亲的牺牲——四十年，每一天都在被反噬侵蚀，却从未让孩子知道
- **收获**：使命升华——他要走母亲走过的路，但这一次不是独自承受
- **感官细节（≥4维）**：
  1. 视觉：月光从岩缝倾泻，照在陈守一脸上干涸的泪痕
  2. 听觉：幽冥兽的嘶吼从远处传来，越来越近
  3. 触觉：玉牌碎块硌着掌心，温热又冰冷
  4. 嗅觉：洞穴里潮湿的泥土味混合着血腥气
  5. 味觉：嘴角血迹的咸腥

### 节拍5（章末转折）
**新困境 + 新机遇**

| 要素 | 内容 |
|------|------|
| **代价（新困境）** | **幽冥兽循血脉波动破岩而入/威胁全面升级** |
| **收获（新机遇）** | **四年使命确立/因果反噬机制完整掌握/握紧玉牌的新决心** |
| **势力影响** | 旧敌趁虚（组织/冥将意识）/新危机降临/备战开始 |
