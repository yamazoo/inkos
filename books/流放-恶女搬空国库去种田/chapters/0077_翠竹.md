# 第77章 翠竹

翠竹下楼的时候，楼梯吱呀响了两声。

苏晚清站在窗边，看着她穿过天井，走向柜台。掌柜正在拨算盘，听见脚步声抬起头，脸上的笑纹堆起来。翠竹问了几句什么，掌柜指了指后院的方向。

那是通往后门的过道，也是通往赵姓客人房间的路。

苏晚清把窗帘放下，在床沿坐下。床板咯吱一声，苏青翻了个身，眼皮动了动，没醒。这两天他睡得越来越沉，是伤在好转的征兆。

她从枕下摸出凤凰玉佩，在指尖转了两圈。玉身温润，是贴身戴惯了的成色。翠竹前日说的那块"苏氏"，想来也在苏青身上。两块合璧，能开苏家密库。密库里有什么？控神毒药配方。

这东西害了苏家满门，如今京城里有人不惜杀人也要拿到它。

脚步声从楼下传来。翠竹回来了，上楼梯的时候脚步比下去时快，像是急着要说什么。

"小姐。"

苏青被惊醒，迷迷糊糊地坐起来。翠竹已经进了门，脸上带着古怪的神色。

"那赵掌柜的不是行商。"翠竹压低声音，"我绕到后院借口讨开水，听见他在房里跟人说话。"

"跟谁？"

"看不见。窗户关着，门也闩着，只听见声音。"翠竹顿了顿，"像是两个人的动静。"

苏晚清眉头一皱。赵掌柜是外地行商，四十来岁，蓄短须，昨日打听成衣铺子。她当时判断可能是巧合，如今看来未必。

"说了什么？"

"只听清一句。"翠竹吞了口唾沫，"说'苏家那丫头跑得快，咱们得换个法子'。"

苏晚清心里一沉。

又是苏家。又是跑得快。这话和隔壁那人说的几乎一模一样。城南砍的那两个，是不是也是因为"跑得快"？

"后面声音太低，听不清了。过了一会儿，门响了，像是有人从后门走了。"

苏晚清沉默片刻。

两个线索，同一个人。赵掌柜打听成衣铺子，是为了摸她逃跑的路线；赵掌柜房里藏人，是为了什么？

"小姐，咱们还打听吗？"

"不打听了。"苏晚清起身，走到窗边，把窗帘掀起一角。天井里没有异常，掌柜还在柜台后面打算盘，街上的人来来往往，看起来都是寻常百姓。

"往后几天，你少下楼。"她放下窗帘，"出门打水我来。"

苏青这时候彻底醒了，揉着眼睛问："出什么事了？"

"没什么。"苏晚清走过去，把被角给他掖了掖，"你伤还没好全，好好养着。"

苏青看了她一眼，欲言又止。

"姐，那玉佩……"苏青犹豫了一下，"要是真有人追杀咱们，不如……不如把玉佩扔了。没了钥匙，那配方就算落在谁手里也打不开。"

苏晚清愣了一下。

这是她没想过的主意。

玉佩是母亲遗物，也是开密库的钥匙。扔掉玉佩，等于断了配方这条线。可反过来想，她手里没有筹码，京城的人会不会觉得她知道什么、藏着什么？济世堂被封，周掌柜的线断了，多少人因为苏家的事掉了脑袋。她一个流放路上跑出来的丫头，凭什么能独善其身？

"扔不掉的。"她摇了摇头，"他们要的东西，我们得自己留着。万一哪天需要用它换条命，没了玉佩，拿什么换？"

苏青沉默了。

窗外传来一阵喧哗。苏晚清走到窗边，看见几个衙役押着个五花大绑的男人往巷口走。那男人低着头，身上衣衫褴褛。

"又砍人？"翠竹凑过来，声音发颤。

"不像。"苏晚清盯着那队人，"砍人是押到城南菜市口，这方向不对。"

衙役们押着人拐进另一条巷子，脚步声渐渐远了。

苏晚清看着空荡荡的巷口，忽然说："收拾东西。"

翠竹愣了："小姐？"

"换客栈。"

"可是小姐，隔壁那人说了会回来，咱们走了不是正好……"

"他回来是摸底的，不是抓人的。"苏晚清打断她，"可赵掌柜不一样。他打听成衣铺子，房里藏着人，开口就是'苏家那丫头跑得快'。他不是摸底，他是定点找人。"

翠竹脸色变了："那咱们现在换地方，他会不会……"

"会。"苏晚清从床下抽出包袱，把昨日收拾好的东西一样样检查，"但咱们不换不行了。"

她走到苏青床边，伸手去扶他。苏青伤还没好全，站不太稳，她便半拖半抱地把他弄下床。

"能动吗？"

"能。"苏青咬着牙，"就是慢点。"

"慢点没事。"苏晚清把包袱背上，匕首别在腰间，"咱们从后门走，绕到城西落星桥，找条船。"

翠竹眼睛一亮："去回春堂找周老伯？"

"不找。"苏晚清摇头，"回春堂被盯上了，去了反而害他。咱们找条乌篷船，在船上待着，等风头过了再说。"

苏青看她："姐，你早就想好了？"

"不算想好。"苏晚清推开窗户，探头看了看外面，"只是觉得不能一直等着。隔壁那人在摸底，赵掌柜在藏人，街上又有人被押走。窝在一个地方不动，迟早被人堵死。"

她翻出窗台，回头对翠竹说："扶着苏青，我接你们。"

后巷没什么人，日光从屋檐缝隙里漏下来，照在青石板上。苏晚清走在最前面，翠竹扶着苏青跟在后面。

走到巷口，她探头看了看。街上人来人往，没有可疑的人。她做了个手势，三人快步穿过街道，拐进另一条巷子。

城西落星桥不远，走快点一炷香就能到。桥下有条小河，连着城外的护城河，乌篷船不少。

"姐。"苏青忽然扯了扯她的袖子。

"怎么？"

"后面有人跟着。"

苏晚清脚步一顿，没有回头，只是放慢了步子，用余光往后扫了一眼。巷子深处，一个穿灰布短打的汉子正不紧不慢地走着，手里提着个竹篮，像是买完菜回家的样子。

可这条巷子两头不靠街市，哪来的买菜的人？

"走快些。"苏晚清低声说。

三人加快脚步，拐进一条更窄的巷子。那巷子七拐八绕，她记得前日侦察时走过。苏青伤还没好全，咬着牙跟上，额头上沁出细汗。

走到巷尾，眼前豁然开朗。落星桥就在不远处，桥下河水清浅，几条乌篷船停在岸边。船家正在船头补网，见有人来，抬头看了一眼，又低下头去。

苏晚清快步走向船家，还没开口，就听见身后传来急促的脚步声。

她回头。灰衣汉子已经追到巷口，手里不再是竹篮，而是一把短刀。

"苏姑娘，别跑了。"

那汉子站在巷口，没有再往前走。他的目光扫过翠竹和苏青，最后落在苏晚清脸上，嘴角扯出一个笑。

"我家主人想见你。"

苏晚清没有说话，手悄悄摸向腰间的匕首。

"姑娘放心，不是要害你。"灰衣汉子把短刀收回腰间，"我家主人说了，只要你肯见一面，之前的事既往不咎。"

"你家主人是谁？"

"见了便知。"

苏晚清冷笑一声："不见呢？"

灰衣汉子叹了口气："那就只好请姑娘走一趟了。"

他往前迈了一步。翠竹挡在苏青身前，脸色发白。苏青想动，被苏晚清一把按住。

"别动。"

苏晚清盯着灰衣汉子，眼珠转了转。

这条巷子两头开口，她在前，他在后，硬闯不是不行，但苏青走不快，翠竹又不会武功，真动起手来讨不了好。

而且他说"之前的事既往不咎"，这话里有话。之前什么事？是苏家的事，还是控神毒药？

"你家主人，在哪儿？"

灰衣汉子见她松口，脸上露出一丝笑意："不远，就在城北清风巷。"

清风巷。苏晚清心里一动。老夫人信里提过，城北清风巷有个书铺，是苏家接应点之一。

可老夫人也说过，那书铺能不能联络，要看情况。如今济世堂被封，周掌柜的线断了，清风巷那书铺还开着吗？

"走。"她对翠竹说，"苏青，扶稳了。"

翠竹愣了一下："小姐，真要去？"

"去。"苏晚清迈步往巷口走，经过灰衣汉子身边时，头也没回，"不带路吗？"

灰衣汉子笑了笑，跟上来："姑娘爽快。"

三人出了巷子，往城北方向走。落星桥在身后越来越远。

苏晚清心里清楚，这一步迈出去，就不知道还能不能回头。可有些事，躲是躲不掉的。隔壁那人在摸底，赵掌柜在藏人，街上有人在抓人，与其被人堵在客栈里，不如主动去看看，这些人究竟想要什么。

她摸了摸腰间的匕首。

想要她的命，没那么容易。想要配方，也得看她答不答应。

清风巷就在前面了。
